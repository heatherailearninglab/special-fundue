Steal these prompts!

Use these prompts, remix them, and make them your own. This page is designed to feel practical and playful at the same time, with real examples visitors can study and adapt.

How to use this page

Copy a prompt. Change the subject, audience, format, or tone. Then compare your result to the example and adjust. This helps visitors learn by doing, not just reading.

Copy-friendlyClassroom readyCreative

Example prompt

create a 1/7 scale commercialized figurine of the characters in the picture, in a realistic style, in a real environment. The figurine is placed on a computer desk. The figurine has a round transparent acrylic base, with no text on the base. The content on the computer screen is a 3D modeling process of this figurine. Next to the computer screen is a toy packaging box, designed in a style reminiscent of high-quality collectible figures, printed with original artwork. The packaging features two-dimensional flat illustrations.

Tip: Visitors can swap in a different pet, person, character, or product idea. They can also change the scale, the environment, or the style.

More prompts visitors can steal

Make a classroom visual

Create a bright, kid-friendly poster that explains [topic] for [grade level] using simple language, strong contrast, and inclusive illustrations. Add a short title, three key facts, and one question at the bottom.

Turn notes into a handout

Turn these notes into a one-page handout for [audience]. Use clear headings, bullet points, plain language, and a friendly tone. End with three takeaways and one reflection question.

Build a better prompt

Improve this weak prompt so it gives clearer results. Add purpose, audience, tone, structure, and any missing details. Then explain what changed and why it should work better.
